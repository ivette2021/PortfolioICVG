Ecommerce

Creating a Marketplace application from scratch involves leveraging diverse technologies such as Firebase integration, map SDK with GPS, real-time chat, and push notifications. Additionally, Google Cloud is utilized, and features include ad creation,account verification, camera/gallery access, password changes, category establishment, and interface enhancements through animations. The implementation covers phone calls, text messaging, user permission management, fragments, activities, and an image library.

Login IG

Practice on Instagram login. This involved working on the interface, business logic, validation of email and password formats, functionality to hide and show the password, as well as the loading screen. This whole process was done using Jetpack Compose with Kotlin, and dependency injection was implemented with Dagger Hilt. In addition, the code was restructured to follow the Clean Architecture pattern and the Singleton design pattern was used in the development.
